# index-obsport-pages

## 项目简介

`index-obsport-pages` 是一个用于归档和发布独立 HTML 页面的仓库。这里存放了多个不特定于某一域名或网站的静态页面资源，适用于页面存档、示例展示或临时发布等场景。

这些页面彼此独立，不依赖外部数据库或后端服务，可通过标准 HTTP 服务器直接访问。

## 目录说明

```
.
├── README.md           # 本文件
├── index.html          # 仓库主页 / 目录页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：所有归档的 HTML 页面均放置于此目录下，文件名清晰标识页面用途。
- `assets/`：存放页面共用的 CSS、JavaScript 或图片资源，避免重复引用。

## 页面归档说明

每个 HTML 页面均为独立文件，可单独访问。页面内容可能包含示例、演示、信息展示或临时工具等。不保证所有页面长期有效或内容更新，仅供参考与归档用途。

若需新增页面，请将文件放入 `pages/` 目录，并确保引用资源路径正确（建议使用相对路径）。

## 维护说明

- 本仓库主要供个人或团队内部归档使用，不提供对外服务承诺。
- 页面内容可能不定期清理或更新，恕不另行通知。
- 欢迎提交 Pull Request 修正页面错误或优化资源引用。
- 如有疑问，可提交 Issue 讨论。
